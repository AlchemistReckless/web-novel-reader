# Web Novel Reader - Feature List

## 📚 Core Reading Features
- [ ] **Infinite Scroll Reading** - Seamless chapter-to-chapter navigation
- [ ] **Paged Reading Mode** - Traditional page-turning experience
- [ ] **Vertical Scroll (Webtoon Mode)** - Manga-style vertical reading
- [ ] **Reading Progress Tracking** - Visual progress bars and percentage
- [ ] **Bookmarking** - Save reading position per novel
- [ ] **Reading History** - Recently read chapters log
- [ ] **Offline Downloads** - Download chapters for offline reading
- [ ] **Auto-Save Progress** - Automatic reading position sync

## 🎨 Reader Customization
- [ ] **Font Selection** - Multiple font options (Serif, Sans-serif, Custom)
- [ ] **Font Size Controls** - Dynamic A- / A+ adjustment
- [ ] **Line Spacing** - Compact, Normal, Relaxed options
- [ ] **Margin Adjustment** - Customizable page margins
- [ ] **Background Themes** - Light, Sepia, Dark, Black (OLED)
- [ ] **Text Alignment** - Left, Justified, Center options
- [ ] **Brightness Control** - In-app brightness slider

## 🔍 Discovery & Search
- [ ] **Global Search** - Search across all sources
- [ ] **Advanced Filter** - Genre, Status, Author, Year filters
- [ ] **Tag Cloud** - Popular tags browsing (Action, Romance, Isekai, etc.)
- [ ] **Latest Releases** - Real-time chapter update feed
- [ ] **Completed Novels** - Filter for finished series
- [ ] **Recommendation Engine** - "You might also like" suggestions
- [ ] **Trending Rankings** - Popular novels leaderboard

## 📖 Library Management
- [ ] **Multi-Source Library** - Aggregate from Novel Updates, Baka-Updates, etc.
- [ ] **Custom Categories** - Create personal collections
- [ ] **Reading Lists** - Plan to Read, Reading, Completed, On-Hold, Dropped
- [ ] **Favorites** - Quick access to loved novels
- [ ] **Update Notifications** - Push notifications for new chapters
- [ ] **Batch Updates** - Refresh entire library for new chapters
- [ ] **Cover Grid/List View** - Toggle display modes

## 🔊 Accessibility & Audio
- [ ] **Text-to-Speech (TTS)** - Android TTS integration
- [ ] **Background Playback** - Listen while app is backgrounded
- [ ] **Voice Customization** - Adjust pitch, speed, voice selection
- [ ] **Sleep Timer** - Auto-stop after set duration
- [ ] **Live Translation** - MLKit/Google Translate integration

## 💬 Social Features
- [ ] **Chapter Comments** - Per-paragraph or per-chapter discussion
- [ ] **Review System** - Star ratings and written reviews
- [ ] **User Profiles** - Reading stats, favorites public/private
- [ ] **Community Forums** - Novel discussion boards
- [ ] **Share Chapters** - Social media sharing

## ⚙️ Technical Features
- [ ] **Multi-Language Support** - EN, PT, ES, JP, CN sources
- [ ] **Local EPUB Import** - Read local files
- [ ] **Backup & Restore** - Cloud or local data backup
- [ ] **Cross-Device Sync** - Reading progress across phones/tablets
- [ ] **Data Saver Mode** - Compress images, reduce data usage
- [ ] **Auto-Download** - Auto-fetch new chapters on WiFi

## 🎮 Advanced Features
- [ ] **Web Scraping Extensions** - Plugin system for new sources
- [ ] **Custom CSS** - Advanced reader styling
- [ ] **Annotation/Highlighting** - Save and export highlights
- [ ] **Dictionary Lookup** - Tap-to-define words
- [ ] **Reading Statistics** - Time read, chapters completed, etc.
- [ ] **Achievement System** - Gamification elements

## Must-Have: Multi-source aggregation, offline downloads, TTS, customizable reader themes
## High Priority: Global search, library categories, update notifications, backup/restore
## Nice-to-Have: Social comments, achievement system, advanced statistics
